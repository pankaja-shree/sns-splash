Slack and Study


A Slash command to manage private groups in your team

Slack and Study helps you to easily manage private study groups in a slack team using slash commands.

What does it do?

For all the private groups that exist in /sns database, any user can:

  • Easily search for a private group belonging to a particular topic.
  • Find the list of all private groups in a team.
  • Easily join any private group.

How to use /sns

/sns set-topic [topic] Sets the topic of the private channel in which the command is executed and adds it to database
/sns show-all Shows all the private study groups in the team
/sns find [topic] Finds the study-group with the given topic
/sns add-to [channel] Shows a button to directly enter the study-group with invite link
