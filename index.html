Slack and Study


A Slash command to manage private groups in your team

Slack and Study helps you to easily manage private study groups in a slack team using slash commands.

What does it do?

Any user can easily search for a private group, find the list of all private groups that exist in /sns database and easily join the private group.

How to use it?

How to use /sns command:

/studybot set-topic [topic] Sets the topic of private channel and adds it to database

/studybot show-all Shows all the private study groups in the team

/studybot find [topic] Finds the study-group with the given topic

/studybot add-to [channel] Shows a button to directly enter the study-group with invite link
